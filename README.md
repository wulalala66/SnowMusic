# SnowMusic

**SnowMusic** 是一款适配 **洛雪（LX）生态音源** 的 Android 音乐播放器。

在应用内可 **在线或本地导入** 符合洛雪规范的 **音源脚本**（`.js`），由内置 **LX 脚本运行时** 参与解析播放地址、歌词等，并与多平台在线能力协同——可以理解为：**以洛雪音源为插件层的本地 / 在线混合播放器**。

本项目 **并非** 洛雪官方客户端；实现上对齐常见洛雪移动端脚本的接口与使用习惯，便于使用社区洛雪系音源资源。

## 下载

请在本仓库 **[Releases](https://github.com/wulalala66/SnowMusic/releases)** 页面下载最新 **`app-release.apk`**。

应用内「检查更新」亦指向上述 Releases。

## 使用提示

- 音源脚本需自行准备；导入、更新与版本展示见应用内 **设置**。
- 请遵守音源服务条款与当地法律法规；第三方脚本请自行甄别安全性。

## 开发与构建

源码与完整文档在私有开发仓库维护；本仓库主要用于 **对外发布安装包** 与版本更新检查。
